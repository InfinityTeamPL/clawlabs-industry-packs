## Jak Janina używa orzecznictwa

1. Przy pytaniu o sprawę **sporną** — `memory_search` w tym katalogu.
2. Cytuj: **sygnatura + teza + podstawa prawna**, zawsze z zastrzeżeniem, że to wskazówka, nie gwarancja.
3. **Nigdy nie zmyślaj sygnatury.** Jeśli nie ma jej w tej bazie — powiedz wprost „nie mam orzeczenia w bazie, sprawdź w SAOS/LEX” i użyj web-search. Fałszywa sygnatura = utrata wiarygodności.
4. Orzecznictwo uzupełnia, nie zastępuje przepisu — najpierw cytat z ustawy, potem teza SN jako wykładnia.
