# Onboarding nowego pracownika — checklist

_Od oferty do 30. dnia pracy._

## Faza 1: Akceptacja oferty → dzień 0 (przed rozpoczęciem pracy)

### Dokumenty od pracownika
- [ ] Kwestionariusz osobowy (wzór)
- [ ] Kopia dokumentu tożsamości (do weryfikacji)
- [ ] Świadectwa pracy z poprzednich miejsc (do wliczenia stażu)
- [ ] Dyplomy / świadectwa ukończenia szkół (wymiar urlopu)
- [ ] Zaświadczenie o niekaralności (jeśli wymagane — branże regulowane)
- [ ] Dokumenty do wliczenia nowego stażu 2026: umowy zlecenia, JDG (CEIDG wypis), zaświadczenia zagraniczne

### Skierowania i badania
- [ ] **Skierowanie na badania wstępne** (przed rozpoczęciem pracy!)
- [ ] Pracownik wykonuje badania — pracodawca opłaca
- [ ] Orzeczenie lekarskie: ZDOLNY → idziemy dalej
- [ ] Jeśli brak orzeczenia → pracownik nie może zacząć

### Przygotowanie umowy i dokumentów
- [ ] Przygotuj **umowę o pracę** (okres próbny / określony / nieokreślony — wg ustaleń)
- [ ] Przygotuj **informację o warunkach zatrudnienia** (art. 29 § 3 KP) — do wręczenia najpóźniej w 7. dniu
- [ ] Przygotuj **klauzulę informacyjną RODO** dla pracownika
- [ ] Sprawdź czy w **regulaminie pracy** nie ma dodatkowych dokumentów do podpisu

### Przygotowanie operacyjne
- [ ] **Biurko / stanowisko pracy** przygotowane
- [ ] **Laptop / komputer** — przygotowany z potrzebnym oprogramowaniem
- [ ] **Konto w systemach** (mail, HRIS, Slack, CRM, VPN) — przygotowane, ale nieaktywne do dnia 1
- [ ] **Telefon służbowy** (jeśli stanowisko tego wymaga)
- [ ] **Karta dostępu / klucze** — przygotowane
- [ ] **Identyfikator / wizytówki** — zamówione

### Komunikacja
- [ ] Powitalny mail do pracownika (co się stanie dzień 1 — godzina, miejsce, kontakt, dress code, co przyniesie)
- [ ] **Poinformuj zespół** o nowej osobie (imię, rola, data startu)
- [ ] **Buddy / mentor** wyznaczony — powiadomienie go o roli
- [ ] Plan dnia 1 przygotowany (powitanie, szkolenia, lunch z zespołem)

## Faza 2: Dzień 1

### Powitanie (rano)
- [ ] **9:00** — przyjęcie pracownika, kawa/herbata, spokój
- [ ] Oprowadzenie po biurze (pomieszczenia, kuchnia, toalety, miejsce BHP)
- [ ] Przedstawienie zespołu (krótkie, zrozumiałe imiona + rola)
- [ ] Stanowisko pracy — pokazanie

### Dokumenty — podpisanie
- [ ] **Umowa o pracę** — podpis 2 egzemplarze (1 pracodawca, 1 pracownik)
- [ ] **Klauzula informacyjna RODO** — podpisany odbiór
- [ ] **Zapoznanie z regulaminem pracy** — podpis
- [ ] **Zapoznanie z regulaminem wynagradzania** — podpis
- [ ] **Zapoznanie z polityką antymobbingową** — podpis
- [ ] **Oświadczenie o zapoznaniu z oceną ryzyka zawodowego** — podpis
- [ ] **PIT-2** — oświadczenie podatkowe
- [ ] Zgoda / oświadczenie o rachunku bankowym
- [ ] Deklaracja PPK (jeśli stosuje)

### Szkolenie BHP wstępne
- [ ] **Instruktaż ogólny BHP** — min. 2h, karta szkolenia do podpisania
- [ ] **Instruktaż stanowiskowy BHP** — min. 2h (biurowe) lub 8h (robotnicze), karta szkolenia
- [ ] **Szkolenie przeciwpożarowe** (jeśli osobne)

### Operacyjne
- [ ] **Aktywacja kont** (mail, systemy, Slack, HRIS)
- [ ] **Wydanie sprzętu** — protokół przekazania (laptop, telefon, klucze)
- [ ] **Karta dostępu / klucz** — wydanie
- [ ] **Lunch z zespołem lub buddym** — integracja, nie biznes

### Zgłoszenie urzędowe
- [ ] **Zgłoszenie do ZUS (ZUS ZUA)** — w ciągu 7 dni
- [ ] **Wpis do ewidencji pracowników**
- [ ] **Założenie akt osobowych** (4 części: A, B, C, D)

---

## Faza 3: Tydzień 1-2

### Szkolenia wewnętrzne
- [ ] **Produkt / usługi firmy** — kogo obsługujemy, co robimy
- [ ] **Procesy** — jak w praktyce działają kluczowe obszary
- [ ] **Narzędzia** — systemy, aplikacje, specyficzne platformy
- [ ] **Komunikacja wewnętrzna** — kanały, konwencje, kiedy ping, kiedy mail

### Cele i oczekiwania
- [ ] **Spotkanie z przełożonym** — pierwsze oczekiwania, cele na 30/60/90 dni
- [ ] **Pisemny plan** pierwszego miesiąca (co ma osiągnąć)
- [ ] **Wyznaczenie buddy'ego / mentora** — osoba, do której pracownik przychodzi z pytaniami

### Integracja
- [ ] Spotkanie z kluczowymi interesariuszami (inne zespoły z którymi będzie pracował)
- [ ] Lunch / kawa z kilkoma osobami z zespołu
- [ ] Dostęp do dokumentacji, wiki, materiałów onboardingowych

---

## Faza 4: Koniec miesiąca (dzień 30)

### Ocena
- [ ] **Spotkanie oceniające** — pracownik + przełożony
- [ ] Przegląd celów z planu 30-dniowego
- [ ] Feedback dla pracownika (co idzie dobrze, co poprawić)
- [ ] Feedback od pracownika (jak się czuje, czego brakuje)
- [ ] Plan na kolejne 30 dni

### Dokumentacja
- [ ] **Zaświadczenie o szkoleniu wstępnym BHP** — w aktach osobowych
- [ ] **Oświadczenie o zapoznaniu z dokumentami** — kompletne
- [ ] **Wpisanie do systemu kalendarza** — kolejne szkolenia BHP, badania okresowe

### Decyzje (przy okresie próbnym)
- [ ] Czy kontynuujemy po okresie próbnym?
- [ ] Jaki rodzaj umowy po próbie (określona / nieokreślona)?
- [ ] Przygotowanie nowej umowy z wyprzedzeniem

## Cheat sheet Janiny: typowe błędy

### 1. Brak badań przed pracą
❌ "Poszedł już pracować, badania zrobi w tym tygodniu"
✅ **NIE DOPUSZCZAMY** do pracy bez orzeczenia medycyny pracy

### 2. Brak szkolenia BHP dzień 1
❌ "Szkolenie przeprowadzimy na początku przyszłego tygodnia"
✅ Szkolenie wstępne **przed** rozpoczęciem pracy lub w 1. dniu

### 3. Spóźnione ZUS ZUA
❌ "Zgłosimy do ZUS pod koniec miesiąca z listą"
✅ **7 dni** od rozpoczęcia pracy to **maksymalny termin**

### 4. Brak potwierdzenia otrzymania klauzuli RODO
❌ Wrzucamy klauzulę na biurko pracownika
✅ Podpisany odbiór w 2 egzemplarzach (akta + pracownik)

### 5. Brak planu celów
❌ "Rozgląda się, zobaczymy za miesiąc"
✅ Pisemny plan 30/60/90 dni — pracownik wie co ma osiągnąć

### 6. Ignorowanie onboardingu zdalnego
❌ "Pracuje z domu, wyśle mu tylko laptopa"
✅ **Pełny onboarding także zdalnie** — wideokonferencja, dostępy, kurier ze sprzętem, mentor na callu

## Akta osobowe — struktura (art. 94 pkt 9a KP)

**Część A** — dokumenty z rekrutacji
- CV, list motywacyjny
- Kwestionariusz
- Świadectwa pracy, dyplomy
- Orzeczenie lekarskie
- Zaświadczenia (niekaralności, jeśli wymagane)

**Część B** — przebieg zatrudnienia
- Umowa o pracę
- Aneksy
- Oświadczenia PIT
- Szkolenia BHP
- Zgody RODO / dodatkowe
- Informacja o warunkach zatrudnienia

**Część C** — rozwiązanie
- Wypowiedzenie / porozumienie
- Świadectwo pracy
- Rozliczenie urlopu / ekwiwalentu

**Część D** — kary porządkowe (jeśli były)

## Narzędzia online do onboardingu (do rozważenia)

- **HRIS z funkcją onboarding workflow** (BambooHR, Personio, HRappka)
- **Dedykowane onboarding apps** (Leapsome, Enboarder, Talmundo)
- **Wewnętrzna baza wiedzy** (Notion, Confluence, Google Site)
- **Welcome box** — koszulka, kubek, mini-gadżet, notes (pozytywne "hej witaj")
