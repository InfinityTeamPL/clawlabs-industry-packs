# Offboarding — checklist rozwiązania stosunku pracy

_Od decyzji / wypowiedzenia do 30 dni po ustaniu zatrudnienia._

## Faza 1: Decyzja / wypowiedzenie

### Przygotowanie (przed wręczeniem wypowiedzenia przez pracodawcę)
- [ ] Sprawdź okresy ochronne (przedemerytalny, ciąża, urlop, chroniony) — **NIE WRĘCZAJ** jeśli są
- [ ] Ustal okres wypowiedzenia (wg stażu łącznego — pamiętaj o zmianach 2026)
- [ ] Przygotuj **pisemne wypowiedzenie** z obowiązkową przyczyną (konkretna, rzeczywista)
- [ ] Przygotuj pouczenie o sądzie pracy (21 dni)
- [ ] Konsultacja ze związkiem zawodowym (5 dni na opinię) — jeśli dotyczy
- [ ] Zaplanuj rozmowę z pracownikiem — nie w piątek wieczorem, nie w obecności innych

### Wręczenie / podpisanie
- [ ] **Wypowiedzenie** — pisemne, z podpisem pracodawcy, z przyczyną, z pouczeniem
- [ ] **Lub porozumienie stron** — jeśli pracownik się zgadza na szybsze rozstanie
- [ ] **Data doręczenia** — odnotowana, podpis pracownika "otrzymałem"
- [ ] Jeśli odmawia podpisu — notatka służbowa + świadkowie

### Komunikacja wewnętrzna
- [ ] **Kierownik zespołu** — wie, planuje przekazanie
- [ ] **IT** — planuje dezaktywację (nie od razu, na dzień X)
- [ ] **Finanse / księgowość** — planuje wypłatę, odprawę, ekwiwalent
- [ ] **Klienci zewnętrzni** — jeśli kontaktowy, planujemy transition communication

---

## Faza 2: Okres wypowiedzenia

### Przekazanie obowiązków
- [ ] **Lista zadań / projektów** przekazywanych — kto co przejmuje
- [ ] **Dokumentacja** — pracownik uzupełnia, porządkuje, tworzy "knowledge transfer"
- [ ] **Klienci** — spotkania wprowadzające z nowym kontaktem
- [ ] **Dostępy** — co kto musi mieć przekazane (hasła zmieniać zawsze, ale uprawnienia transferować)

### Prawa pracownika w okresie wypowiedzenia
- [ ] **Urlop do wykorzystania** — ustalenie terminów w okresie wypowiedzenia LUB ekwiwalent
- [ ] **Zwolnienie na poszukiwanie pracy** (art. 37 KP):
  - 2 dni robocze przy wypowiedzeniu 2-tygodniowym lub 1-miesięcznym
  - 3 dni robocze przy wypowiedzeniu 3-miesięcznym
- [ ] **Wynagrodzenie** — pełne, bez zmian

### Exit interview
- [ ] **Rozmowa z pracownikiem** (1-na-1, bez presji)
  - Dlaczego odchodzi (jeśli sam zrezygnował)
  - Co działało, co nie działało
  - Co byśmy zmienili, gdyby mogli wrócić
  - Referencje / plany
- [ ] **Anonimowa ankieta** — opcjonalnie, jeśli woli
- [ ] **Notatka** w aktach — konstruktywna, bez ocen subiektywnych

---

## Faza 3: Ostatni dzień pracy

### Zwroty
- [ ] **Laptop / komputer**
- [ ] **Telefon służbowy**
- [ ] **Klucze / karty dostępu**
- [ ] **Identyfikator / karta wizytówki firmowej**
- [ ] **Karnet sportowy / karta medyczna** (jeśli firmowa)
- [ ] **Materiały firmowe** (książki, dokumenty, próbki)
- [ ] **Samochód służbowy** (jeśli dotyczy)
- [ ] **Karta paliwowa / służbowa**
- [ ] **Protokół przekazania** — podpis obu stron

### Dezaktywacja dostępu (IT)
- [ ] **Konto e-mail** — przekierowanie na kierownika lub wyznaczoną osobę (okres 30-90 dni)
- [ ] **Systemy firmowe** (CRM, ERP, HRIS, Slack) — dezaktywacja kont
- [ ] **VPN / dostępy zdalne** — odebrane
- [ ] **Hasła do wspólnych kont** — zmienione
- [ ] **Karty dostępu do biura** — deaktywowane

### Dokumenty do wręczenia pracownikowi
- [ ] **Świadectwo pracy** — niezwłocznie, max. 7 dni
- [ ] **PIT-11** — do końca lutego następnego roku (po zakończeniu)
- [ ] **Zaświadczenie o zarobkach** (jeśli prosi — np. do ZUS, banku, urzędu)
- [ ] **Zaświadczenie o szkoleniu BHP** (kopia)
- [ ] **Referencje** — jeśli pracownik o nie prosi i pracodawca się zgadza

### Ostatnia rozmowa
- [ ] Podziękowanie
- [ ] Ostatnia wypłata ustalona (kiedy, gdzie, co)
- [ ] Konto bankowe do wypłaty ekwiwalentu — potwierdzone

---

## Faza 4: Po ustaniu zatrudnienia

### Rozliczenia (ekwiwalent: termin wypłaty wynagrodzenia albo 10 dni po ustaniu — art. 171 §4–5 KP)
- [ ] **Wynagrodzenie** za przepracowany okres — w normalnym terminie
- [ ] **Ekwiwalent urlopowy** — obliczony i wypłacony
- [ ] **Odprawa** (jeśli przysługuje — ustawa o zwolnieniach grupowych)
- [ ] **Nagrody, premie roczne** (jeśli regulaminowo się należą)
- [ ] **Rozliczenie PPK** — wypłata / transfer / kontynuacja wg decyzji pracownika

### Wyrejestrowanie
- [ ] **ZUS ZWUA** — wyrejestrowanie w ciągu **7 dni**
- [ ] **Przekazanie danych do ZUS o okresie składkowym** (dla pracowników przed 2019 — RIA)

### Akta osobowe
- [ ] **Część C** — wypowiedzenie / porozumienie, świadectwo pracy, rozliczenia
- [ ] **Retencja** — 10 lat po ustaniu zatrudnienia (pracownicy od 2019)
- [ ] **Archiwum** — przekazanie do archiwum fizycznego / elektronicznego

### Komunikacja
- [ ] **Out-of-office** na skrzynce mailowej ze wskazaniem kontaktu
- [ ] **Aktualizacja strony internetowej** (usunięcie z sekcji "zespół")
- [ ] **Aktualizacja LinkedIn** firmy (jeśli potrzebna)
- [ ] **Pożegnanie zespołu** (mail lub spotkanie)

---

## 30 dni po — follow-up

### Zadania
- [ ] **PIT-11** wystawiony (jeśli termin się zbliża)
- [ ] **Zaświadczenia** — jeśli przychodzą żądania (bank, ZUS, kolejny pracodawca)
- [ ] **Referencje** — gdy pojawia się prośba
- [ ] **Pozostałe rozliczenia** — nadgodziny, ekwiwalent, niewypłacone składniki

---

## Offboarding specjalny — okoliczności

### Śmierć pracownika
- [ ] **Stosunek pracy wygasa** (art. 63¹ KP) — z dniem śmierci
- [ ] **Odprawa pośmiertna** (art. 93 KP) — dla rodziny, w wysokości:
  - 1-miesięcznego wynagrodzenia — staż < 10 lat
  - 3-miesięcznego — staż 10-15 lat
  - 6-miesięcznego — staż > 15 lat
- [ ] **Świadectwo pracy** — dla rodziny
- [ ] **PIT-11** — za okres pracy

### Długotrwała niezdolność do pracy (art. 53 KP)
- Rozwiązanie umowy bez wypowiedzenia **po okresie ochronnym** (zwykle 3 mies. zasiłku + 3 mies. L4 + 3 mies. świadczenia rehabilitacyjnego)
- Dokumentacja: orzeczenie lekarskie o braku możliwości powrotu do pracy
- Wypowiedzenie + świadectwo pracy

### Rozwiązanie z winy pracownika (art. 52 — "dyscyplinarka")
- **1 miesiąc** na decyzję od dowiedzenia się
- Pisemna przyczyna (kradzież, stawiennictwo pod wpływem, etc.)
- **Bez wypowiedzenia, bez odprawy** — ale z rozliczeniem za przepracowany czas
- Konsultacja ze związkiem (jeśli pracownik jest członkiem)

### Likwidacja zakładu pracy
- Wszyscy pracownicy — wypowiedzenia
- **Odprawy** zgodnie z ustawą o zwolnieniach grupowych
- **Ochrona nie działa** (art. 41¹ KP)
- Cały harmonogram "zamykający"

## Cheat sheet: terminy offboardingowe

| Zadanie | Termin |
|---------|--------|
| Świadectwo pracy | **7 dni** |
| Ekwiwalent urlopowy | termin wypłaty wynagrodzenia; jeśli minął przed rozwiązaniem — **10 dni** po ustaniu (art. 171 §4–5 KP) |
| Wypłata ostatniego wynagrodzenia | Standardowy termin (do 10. dnia mies.) |
| Wyrejestrowanie ZUS (ZWUA) | **7 dni** |
| Archiwizacja akt | Bezpośrednio po offboardingu |
| PIT-11 | **Do końca lutego** następnego roku |
| Retencja akt | **10 lat** po ustaniu (od 2019) |
